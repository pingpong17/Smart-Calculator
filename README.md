# Smart-Calculator

A Python program that implements a calculator, similar to the Python Shell. It supports addition, subtraction, multiplication, integer division. It also supports variables.

Пример использования:
```bash
> 2 * 2
4
> 1 + -2 + 3
2
> 2 * (2 + 2)
8
> x = 3
> y = 4
> z = x * x + y * y
> z
25
> 100 / (z - 25)
Division by zero
```
